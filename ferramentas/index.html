Ferramentas

Ferramentas que serão utilizadas na disciplina. Todas são gratuitas.

Bibliotecas

  • phaser.io: Engine para programação de jogos web mais elaborados e com grande suporte a plugins e ferramentas externas.
  • jQuery: Biblioteca com utilidades para manipulação de HTML em Javascript.

Online

  • Codepen -> Editor online para desenvolvimento web (cliente). Prático para testes e experimentos.
  • Phaser Sandbox -> Editor online para experimentar com a engine Phaser.

Standalone

  • Node.js -> Plataforma para a criação de software em Javascript (servidor e cliente).
  • Brackets -> Editor de código com boas features para preview de código Javascript e desenvolvimento web no geral.
  • WAMP Server -> Programa que cria um servidor básico local para testes.
  • Github For Windows -> Programa para facilitar o gerenciamento de repositórios e desenvolvimento em Windows.
