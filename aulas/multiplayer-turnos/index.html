Multiplayer por turnos

Para exemplificar o desenvolvimento de um jogo online multiplayer por turnos, vamos implementar de forma básica o jogo Othello. Principais características dessa implementação estão detalhadas abaixo.

Servidor

  1. Utilizar websockets para estabelecer canais de comunicação;
  2. Manter todo o estado do jogo e distribuir essa informação aos clientes;
  3. Permitir a criação de partidas simultâneas;
  4. Distribuir novos jogadores automaticamente;

Cliente

  1. Organizar interação do jogador baseado no estado do tabuleiro;
  2. Renderizar o tabuleiro e comunicar fluxo do jogo;
