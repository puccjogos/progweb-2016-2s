Oficina de jogos coletivos

Vamos dedicar a aula de hoje a trabalhar nos projetos de jogo coletivo, preparando as ideias e configurando o básico do projeto. Vamos seguir as seguintes etapas:

  1. Definir dupla.
  2. Decidir quais features vai incorporar: persistência ou integração com serviços.
  3. Preparar SGDD do jogo.
  4. Aprovar ideia.
  5. Preparar estrutura de comunicação entre cliente e servidor.
  6. Criar projeto de Node e preparar sua estrutura básica.
